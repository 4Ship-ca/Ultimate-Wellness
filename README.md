# Ultimate Wellness v2.2.1

AI-powered wellness tracking with multi-user support, camera, recipe intelligence, and cloud-ready architecture.

## Quick Start

1. Open `index.html` in a web browser
2. App works offline (no server needed)
3. Data stored in browser IndexedDB

## Features

- Multi-user with authentication
- Food/exercise/sleep/water tracking
- Camera with zoom, flash, video (20s)
- AI recipe intelligence
- Zero-point foods (180+ items)
- Email templates
- Admin panel
- Session persistence
- Cloud-sync ready

## Admin Panel

Open `admin-complete.html` for full data management.
